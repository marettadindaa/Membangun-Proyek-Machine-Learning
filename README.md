# 📊 Machine Learning Project - DBS Coding Camp

## 📌 Project Description

This project is part of the **Coding Camp powered by DBS Foundation**, specifically for the **Machine Learning for Beginners** course.

The project consists of two main tasks:

1. **Clustering** to group data into meaningful segments
2. **Classification** to predict target classes based on clustering results

---

## 🎯 Objectives

* Understand unsupervised learning (Clustering)
* Understand supervised learning (Classification)
* Apply machine learning workflow from preprocessing to evaluation
* Build and evaluate machine learning models

---

## 🧠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn
* Jupyter Notebook

---

## 📂 Dataset Used

For this project, we are using a modified version of the **Bank Transaction Dataset for Fraud Detection** from Kaggle. The dataset is specifically adjusted to meet the project assessment criteria.

---

## 🔍 Project Workflow

### 1. Clustering

* Perform data preprocessing
* Apply clustering algorithm (e.g., K-Means)
* Analyze cluster results
* Generate new labeled dataset for classification

### 2. Classification

* Load dataset from clustering results
* Perform feature encoding (if needed)
* Split data into training and testing sets
* Train classification model (Decision Tree)
* Evaluate model performance

---

## 🤖 Models Used

* Clustering: K-Means 
* Classification: Decision Tree

---

## 📈 Results

* Successfully grouped data into clusters
* Classification model built using clustering output
* Model evaluation performed using accuracy metric

---

## 🚀 How to Run

1. Open the notebooks using Jupyter Notebook or Google Colab
2. Run all cells sequentially
3. Ensure dataset files are in the correct directory
4. Use the dataset version from **Google Drive** for consistent results

---

## 📝 Key Insights

* Clustering helps uncover hidden patterns in data
* Classification model can leverage clustering results for prediction
* Proper preprocessing significantly improves model performance
